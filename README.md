## 開発ルール
- 自分のタスクは、ブランチを切ってから ../Assets/Member 直下に自分の名前のフォルダを作成してから取り掛かる。
- 各メンバーのブランチ名は以下の通り。 <br> 
  feature/自分の名前/自分の作業箇所
- README ファイル及び gitignore ファイルを編集する場合は、必ず main ブランチで行うこと。<br>
  また、その旨をメンバー全員に通達する事。
- メンバーは報告、連絡、相談を徹底する。
